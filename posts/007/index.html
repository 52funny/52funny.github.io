docker 安装

通过命令安装
wget -qO- https://get.docker.com/ | sh
安装完毕后将普通用户加入到 docker 用户组（为的是让 docker 能让普通用户使用）
sudo gpasswd -a ${USER} docker
然后重启 docker
sudo systemctl restart docker
然后刷新用户组
newgrp docker
