Bilibili 获取 ip

  • 返回值

  "code": 0,
  "message": "0",
  "ttl": 1,
  "data": {
    "addr": "120.243.220.168",
    "country": "中国",
    "province": "安徽",
    "city": "宣城",
    "isp": "移动",
    "latitude": 30.941999,
    "longitude": 118.746819,
    "zone_id": 4407456,
    "country_code": 86
  }
}

返回值data.addr就是 ip 信息
